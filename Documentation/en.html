GrindMachineII-AHK help

Script to use Grind Machine II VST with screen readers.


    This script maybe be working with any DAW, was tested with:

  • Grind Machine II v1.6.
  • Reaper 6.11.
  • Windows 10 x64.

Licence GPL-3.0

Source code and last releases:

https://github.com/crisspro/grindmachineII-ahk

Usage

  1. Run GrindMachineII-AHK
  2. Open Grind Machine II in your DAW.
  3. Press the hotkeys of script.
  4. Can close the script from system tray

Hotkeys

Hotkey Description
M Open the presets menu and another options.
S Next preset.
A Back preset.
F1 Open help documentation.
CTRL+q close the script.
