Ayuda de GrindMachineII-AHK

Script para usar el plugin vST Grind Machine II con lectores de pantalla.

probado con:

  • GrindMachineII 1.6
  • Reaper 6.11
  • Windows10 64bits

Licencia GPL-3.0

Código fuente y últimas versiones:

https://github.com/crisspro/grindmachineII-ahk

Cómo usarlo

  1. Ejecuta GrindMachineII-AHK.
  2. abre Grind Machine II desde tu secuenciador.
  3. Usa los atajos de teclado del script.
  4. Puedes cerrar el script desde la bandeja del sistema

Atajos de teclado

Atajo Descripción
A Cambia al preset anterior
S Cambia al siguiente preset.
M Despliega el menú con los presets y otras opciones.
F1 Abre la Ayuda del script
CTRL+q Sale del script
